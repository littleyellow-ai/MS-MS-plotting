# MS-MS-plotting
Code accompanying the paper "Source-Specific Dynamics of Emerging Contaminants in a Megacity River Basin Revealed by Target and Nontarget Analysis" by Tianchang Huang et al.

Files provided:
A.MS/MS-plotting.R: The main script to plot the comparative spectra. Before executing this code, you need to organize the peak list of each compound in the format of example.txt, and name the files after the corresponding compound names. Finally, place all these txt files into a single folder.

B.example.txt: Example input file containing the peak lists of samples and standards; Each peak is represented in the format of "m/z intensity"
